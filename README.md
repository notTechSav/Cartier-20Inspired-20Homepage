# Cartier-Inspired Homepage

Luxury web application built with modern web standards and refined design principles.

## Design System
- [Luxury Web Standard v1.1](docs/Luxury-Web-Standard.md) - Complete build + experience specification

## Tech Stack
- React + TypeScript
- Tailwind CSS (token-based)
- Vite
- TanStack Query

## Quick Start
```bash
npm install
npm run dev
```

## Build
```bash
npm run build
```

## Performance
- Bundle split by vendor
- Lazy-loaded routes
- Optimized images (AVIF/WebP)
- CSS purging enabled

## Contributing
Please reference the [Luxury Web Standard](docs/Luxury-Web-Standard.md) for all design and implementation decisions.
